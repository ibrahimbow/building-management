# 🏢 Building Management Platform

A modern building management platform that enables property managers and residents to interact within a shared residential environment.

Managers can publish official announcements, while residents can communicate, share information, ask for help, and collaborate through community-driven features.

The platform is built using a modern microservices architecture to demonstrate enterprise-grade backend engineering, scalable system design, and DevOps practices.

---

# 🚀 Tech Stack

* Java 21
* Spring Boot
* Spring Security (JWT)
* JUnit 5, Mockito, Testcontainers
* JPA / Hibernate
* PostgreSQL
* Apache Kafka (event-driven communication)
* Angular
* Docker & Docker Compose
* GitHub Actions & Jenkins
* Prometheus & Grafana
* Kubernetes & Terraform

---

# 🧩 Microservices

* **bm-gateway-service** – API Gateway, routing, and security
* **bm-auth-service** – Authentication & authorization
* **bm-building-service** – Buildings and tenant membership management
* **bm-announcement-service** – Official building announcements
* **bm-share-and-help-service** – Community Share & Help posts
* **bm-notification-service** – Kafka-based realtime notifications
* **bm-file-service** – File and image uploads
* **bm-frontend-angular** – Angular frontend application
* **bm-infra** – Docker, Kubernetes, and Terraform infrastructure configuration
